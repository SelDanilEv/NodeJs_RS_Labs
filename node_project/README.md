params in command line 
* (-s || --shift) (int) example -s 2 
* (-i || --input) (str path to input file) example -i ../files/input.txt 
* (-o || --output) (str path to output file) example -o ../files/output.txt 
* (-s || --action) (decode | encode) example -a decode
